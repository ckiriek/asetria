# Project Overview - Asetria
ЦА: CRO, биотех-стартапы, исследовательские центры 2-4 фазы.
Базовый сценарий: проект - загрузка - извлечение - генерация - валидация - версия/аудит - экспорт.

Ценности для клиента:
- Скорость и соответствие регуляторке
- Единый источник правды для всех документов
- Снижение ручных правок и риска несоответствий
