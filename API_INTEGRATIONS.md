# API Integrations

| API | Purpose | Auth | Cost |
|---|---|---|---|
| ClinicalTrials.gov | Аналоги КИ, дизайн, результаты | none | free |
| PubMed/Entrez | Публикации и evidence base | none | free |
| openFDA | AE/SAE и safety profile | API key (free) | free |
| WHO ICTRP | Глобальные реестры КИ (CSV dump) | none | free |
| MedDRA subset | Термины AE/SAE для маппинга | local CSV | free subset для MVP |
| EMA SPOR/RMS | Справочник организаций | token | free (optional) |
